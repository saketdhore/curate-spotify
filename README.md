🎧 Curate Spotify
An interactive Spotify analytics dashboard that visualizes users' top songs, artists, and genres across time.

📌 Project Overview
Curate Spotify is a full-stack web application designed to give users deep insights into their listening habits. Leveraging Spotify's API and OAuth authentication, the app fetches real-time and historical user data, presenting it in an intuitive and visually appealing dashboard.

This project focuses on transforming raw streaming data into personalized insights through clean UI and scalable backend architecture.

💡 Features
🔐 OAuth Authentication via Spotify

📊 Analytics Dashboard to explore top songs, artists, and genres

🕐 Support for multiple time ranges (last 4 weeks, 6 months, all time)

🎨 Beautiful, responsive UI built with Tailwind CSS

⚙️ Scalable backend with Express.js for handling API requests

🧠 Global state management with Redux

🛠 Tech Stack
Frontend
React

Tailwind CSS

Redux

Chart.js (if used for visualizations)

Backend
Node.js + Express.js

Spotify Web API

OAuth 2.0 Authentication
